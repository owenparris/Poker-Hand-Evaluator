# Poker Equity Calculator

A fast poker equity calculator for evaluating hand strengths.

## Overview

This project computes the probability of winning (equity) for any two given hands.

## Key Features

* Supports arbitrary hands and situations (preflop/postflop)
* Calculates win/tie/loss probabilities
* Designed to handle the large numbers of scenarios efficiently

## Approach

* Enumerates all possible card combinations
* Focused on reducing redundant calculations and improving runtime

## Results

* Produces accurate equity estimates across a wide range of scenarios
* Efficient enough to evaluate large numbers of hands
* Useful for analysing expected value in both tournament and cash game spots

## Tech

* Python

## Notes

The focus of this project was on efficient computation and applying probability to real decision-making situations.
